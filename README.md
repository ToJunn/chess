## Chess Game with Stockfish AI

This is a simple chess game built with Python using Pygame for the graphical interface and the Stockfish chess engine for AI. 

### Features:

- **Playable chess:** Move pieces, capture opponent's pieces, and try to checkmate!
- **Stockfish AI:** Play against the powerful Stockfish engine at different difficulty levels.
- **Adjustable difficulty:** Change the engine depth in the input box to adjust the AI's thinking time and strength. 
- **Side switching:** Choose to play as either white or black at any time.
- **Game restart:** Easily reset the game to the starting position.
- **Visual feedback:**  Highlights selected squares and legal moves for clarity. 
- **Game over detection:** Displays the winner or a draw when the game ends.

### Requirements:

- Python
- Pygame
- Python Chess library 
- Stockfish chess engine
